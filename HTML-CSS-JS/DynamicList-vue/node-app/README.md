### Install VueJS:
1. Download [Node.js](https://nodejs.org)
2. open CMD (Admin)
3. install yarn with ``npm install yarn --global``
4. npm init -y
5. npm install vue --save
6. npm install -g @vue/cli
7. vue create [application-name] > Default preset
8. coding, coding, coding...

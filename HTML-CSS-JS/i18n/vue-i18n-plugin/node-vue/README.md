# i18n

## About this example
- It works with Vue + Vue CLI + [i18n plugin](https://kazupon.github.io/vue-i18n/)
- To install the i18n plugin run `vue add i18n`
- Source: https://www.youtube.com/watch?v=rRhBNfn8dEs
